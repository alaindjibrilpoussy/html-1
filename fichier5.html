Formolaire

Un formulaire est un objet de base de données que vous pouvez utiliser pour créer une interface utilisateur permettant aux utilisateurs d’entrer et de modifier vos données. Les formulaires contiennent souvent des contrôles qui effectuent différentes tâches. Si vous pouvez créer une base de données sans utiliser de formulaires, il vous suffit de modifier vos données dans la table, la plupart des utilisateurs préfèrent utiliser des formulaires pour afficher, entrer et modifier les données contenues dans les tables. Les formulaires vous permettent également de contrôler le mode d’interaction des autres utilisateurs avec les données dans la base de données. Par exemple, vous pouvez créer un formulaire qui n’affiche que certains champs et ne permet d’exécuter que certaines opérations. Cela permet de protéger les données et de réduire les erreurs de saisie de données possibles. Cet article propose un autour de ressources allant des bases des formulaires Access à la création et la personnalisation des formulaires. Commencez par une vue d’ensemble des bases de données. Nous allons nous concentrer ensuite sur les formulaires avec l’introduction des formulaires. Avant de créer un formulaire, jetez un coup d’œil aux contrôles, une fonctionnalité essentielle de la plupart des formulaires. Les contrôles permettent aux utilisateurs d’entrer, de modifier ou d’afficher des données de votre base de données. Il peut être utile de connaître les différents types de contrôles disponibles dans Access lors de la personnalisation de vos formulaires. Pour en savoir plus sur les contrôles, voir Présentation des contrôles.
